What is PaintKit?

PaintKit is a toolset for blender artists interested in easily creating stylized ‘painterly’ looks in their projects. PaintKit includes a growing variety of procedural group tools in both the shader editor and geometry nodes graph to easily create paint-like textures and effects.

Getting Started

Upon installing PaintKit for the first time, make sure to restart blender. After PaintKit has been properly installed, a new panel with be added to your N-Panel in the 3D viewport. PaintKit has two primary functions to be aware of; the Shader Groups, and the Brush Generator.

Brush Generator

The PaintKit Brush Generator is a simple menu that allows you to add brushstroke cards to any object. Simply select your object and click Add Brush. Adding a Brush creates a new object parented to the original, and uses the original as input for the geometry node tools.

Add Brush

Add Brush takes all objects selected, and distributes card geometry across the object. These cards can then be textured normally with basic UVs, or can be given a custom alpha to look like brushstrokes. The paint cards will use the same material as their parent/target object

Add Guided Brush BETA

Add Guided Brush allows for the creation of brushstrokes with directions influence by guide curves, which can be selected in the N-Panel controls. Brushstrokes sample the guide closest to them and extrude the strokes according to the nearest curve tangent. Best used in combination with the Shrinkwrap Toggle

Add SimpleShell

Add SimpleShell creates a shell of the original object using an inverse-hull method. Ideal if you are looking to increase performance as it is much less resource intensive. Using the SimpleShell nodegroup, you have access to masking and material options for the shell.

Remove Brushstrokes

Remove brushstrokes removes all brushes and shells from the selected objects

Shrinkwrap Toggle

Adds a shrinkwrap modifier to the brush. The modifier is placed after the brush generation but before the brush noise settings. Offset can be controlled by the Expand option in the Panel

Hide

hides the brushes on the selected object

Hide all

hides all brush objects in the scene

Shader Groups

SketchToon

Eevee-only shader group made to mimic sketchy-pencil and paint like effects.

CToon Diffuse

Diffuse shader built with the default Toon BSDF, with additional controls and shading levels

CToon Glossy

Glossy Variant of CToon Diffuse. Allows for a shinier look and includes controls for metalness. Metalness control roughly approximates tinted reflections by increasing the saturation of the two upper shading levels.

Normal Painter

Uses a voronoi texture to divide the object normals into discrete sections, creating a more stylized look without having to manually paint different shading areas.

  • Paint size
    • controls the size of the normal ‘sections’
  • Paint Smooth
    • smooths the boundaries between each section
  • Paint Roughen
    • uses a Noise Texture to displace the edges of the sections to roughen up the faceted look of the sections edges
  • Roughen Scale
    • adjusts the scale of the Noise Texture that displaces the section edges
  • Custom Texture
    • Allows for the input of any texture to drive the displacement of the normal sections
  • Custom Offset
    • Strength of the displacement from the custom texture
  • Bump Height, Bump Strength, Bump Distance, Normal Map
    • These controls are the typical controls used in blender’s Bump map and Normal map node
  • Use Custom Normal
    • switch to control whether the group uses the custom normal input or the default for object shading
  • Custom Normal
    • Inputs custom normals to the group as an override for the default normals. Requires input normals to be in world space to operate properly.

NP Controller

Allows for control over Normal Painter to be handled by a simple nodegroup, which can then be copied and instanced into any other material. This helps with having one global controller for all materials using the Normal Painter node
